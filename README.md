# 💬 شات سريع - Real-time Chat

مشروع محادثة فورية بـ Node.js + Socket.io

## 🚀 تشغيل المشروع

```bash
# 1. تثبيت المكتبات
npm install

# 2. تشغيل السيرفر
npm start

# أو للتطوير مع nodemon
npm run dev
```

ثم افتح المتصفح على: **http://localhost:3000**

---

## 📁 هيكل المشروع

```
chat-app/
├── server.js          ← السيرفر الرئيسي (Express + Socket.io)
├── package.json
└── public/
    └── index.html     ← الواجهة الأمامية (HTML + CSS + JS)
```

## 🔧 الـ Events المستخدمة

| Event      | الوصف                        |
|------------|------------------------------|
| `join`     | المستخدم يدخل باسمه          |
| `message`  | إرسال/استقبال رسالة          |
| `typing`   | مؤشر الكتابة                 |
| `users`    | قائمة المستخدمين المتصلين    |
| `system`   | رسائل النظام (دخل/خرج)       |
